# notes-app
 notes app using NodeJs

 commands to run the code 
 
 1- to add a note
  node .\app.js add --title="YOUR Title" --body="YOUR BODY"

2-  to remove a note
    node .\app.js remove --title='YOUR TITLE'

3- To list all The notes
    node .\app.js list

4 - To read a note
    node .\app.js read --title='YOUR TITLE'
